程序介绍：
这是一个类似商品购物的小程序
开发环境：
Windows 7 X64 + Python3.5 + Pycharm5
运行环境：
Windows 7 X64 + Python3.5 + Pycharm5
其他未测
项目文件：
shop.py login.txt lock.txt
主要功能：
主要实现了用户在程序中购买商品的功能，并且做了以下判别：
1、用户注册 初始金额0
2.判断用户金额是否买得起
3.金额不足删除购物车中的商品或者充值
4.用户购买商品余量自减
5、购买商品后余额自减或者充值后退出余额自价，下次登录余额不变
以上是小程序实现的几个功能
