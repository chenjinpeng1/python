    使用方法：

    作业一：

    运行作业一主文件index.html，进入主界面


    点击后面的的编辑按钮打开编辑对话框


    输入正确点击确定提交

    如果输入错误会出现如下图所示错误提示


    可对错误的地方做出修改，直到正确位置

    规则：

    • 主机名：必须填写
    • IP地址：必须填写，并且符合IP地址规范如：192.168.0.1
    • 配置：必须填写
    • 端口：必须填写、必须是数字且大于0小于65535

    作业二：

    运行作业一主文件index2.html，进入主界面

    勾选表格第一列的复选框选择要编辑的主机


    或者可以通过全选、反选这两个按钮进行快速选择


    选中要编辑的主机后点击编辑模式按钮进入编辑模式


    进入编辑模式后即可批量对主机信息进行编辑


    点击取消按钮取消编辑


    编辑结束后点击保存按钮保存编辑的信息


    输入正确点击保存后主机信息发生变化的将会别提交到后台，主机信息发生变化的将会别提交到后台，主机信息发生变化的将会别提交到后台（重要的事情说三遍）

    提交成功后会出现对话框，点击确定后退出编辑模式


    如果啥都没有修改就提交，那不好意思，会对你提出警告哦


    如果输入错误会出现如下图所示错误提示


    点击确定后，出现错误的地方将会出现红色的框，如下图所示


    可对错误的地方做出修改知道正确位置

    规则：

    • 主机名：必须填写
    • IP地址：必须填写，并且符合IP地址规范如：192.168.0.1
    • 配置：必须填写
    • 端口：必须填写、必须是数字且大于0小于65535
